# 嘉豪的个人网站

这是一个使用 Next.js 和 Tailwind CSS 构建的个人网站，用于展示我的项目、博客和联系方式。

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- React Icons
- Framer Motion

## 功能

- 响应式设计
- 深色模式支持
- 项目展示
- 博客文章
- 联系表单

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 构建

```bash
# 构建静态网站
pnpm build
```

## 部署

该网站通过 GitHub Actions 自动部署到 GitHub Pages。每当代码推送到 main 分支时，都会触发部署流程。
